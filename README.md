# Hosting Your Scavenger Hunt on GitHub Pages

## Steps:
1. Go to [GitHub](https://github.com) and create a free account.
2. Create a new repository (e.g., `holiday-scavenger-hunt`).
3. Upload the file `scavenger_hunt_10_days.html`.
4. Commit the changes.
5. Go to **Settings** > **Pages**.
6. Under **Source**, select `main` branch and `/ (root)` folder.
7. Click **Save**.
8. Your site will be live at: `https://yourusername.github.io/holiday-scavenger-hunt`.

Share the link with your team!
